Spell Checker implemented in Java using standard NLP techniques. 

The following methods will be implemented:
1) Knowledge based approach : Edit distance as a metric with bigram probabilities to resolve ties.
2) Generative approach : Using confusion matrices, alignment and pronunciation dictionaries.
3) Discriminative approach

The repository is an eclipse project. Import in eclipse and compile and run Framework.java
Alternatively compile each of the .java files in src/ directory and run Framework.java
